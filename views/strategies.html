Automated Trading Strategies

Configure algorithms that monitor market conditions and suggest or execute trades based on technical indicators. Strategies only run while this browser tab is open.

How Strategies Work
  • Backtesting: See how a strategy would have performed historically before enabling it live.
  • Paper-first: Run strategies on your paper account to validate before using real money.
  • Browser-based: Strategies monitor the market while this tab is open. For scheduled execution, GitHub Actions can be configured later.
  • Safety rails: All strategies respect your risk limits from Settings (position size, daily loss limit, etc.).
