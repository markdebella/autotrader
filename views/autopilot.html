Setup needed for scheduled trading On-demand Run a cycle now works already. Unattended scheduled runs need a one-time cloud setup (all free-tier). In Google Cloud Shell (project autotrader-497920):
  1. Enable the Firestore & Cloud Scheduler APIs.
  2. Create a Firestore database (Native mode).
  3. Grant the backend service account Firestore access (roles/datastore.user).
  4. Create a scheduler service account and redeploy the backend with its email as CRON_SA_EMAIL.
  5. Create the Cloud Scheduler job that calls /api/autopilot/scheduled-run.
