Alpaca Brokerage Connection

Your Alpaca API keys are never entered or stored in this app. They live only in Google Secret Manager and are used solely by the AutoTrader backend service, which this dashboard reads from. The browser holds no trading keys.

Paper vs. live trading is set on the backend service (currently paper mode). To change or rotate keys, update the secrets in your Google Cloud project — see SECURITY.md and docs/gcp-setup.md in the repo.

Backend:

Trade Ideas

On the Ideas tab, Generate ideas asks the backend for AI-written suggestions (via Claude), grounded in your watchlist, account, and risk limits.

The Claude key lives only in Google Secret Manager — never in the browser. If Claude is ever unavailable, the backend automatically falls back to a free, deterministic rules engine so you always get ideas.

Approve an idea and press Place paper order — the backend submits it to Alpaca in paper mode. No copy/paste, nothing to install.

Managing Your API Keys

Every key lives only in Google Secret Manager — never in this app, your browser, or Drive. To rotate or replace one, run its command in Google Cloud Shell (project ), then run the refresh command so the backend picks up the new version.

Then refresh the backend (after any key change)
The service caches secrets per instance; this rolls a new revision so it re-reads them.
