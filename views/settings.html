Alpaca Brokerage Connection

Your Alpaca API keys are never entered or stored in this app. They live only in Google Secret Manager and are used solely by the AutoTrader backend service, which this dashboard reads from. The browser holds no trading keys.

Paper vs. live trading is set on the backend service (currently paper mode). To change or rotate keys, update the secrets in your Google Cloud project — see SECURITY.md and docs/gcp-setup.md in the repo.

Trade Ideas

On the Ideas tab you can generate trade suggestions two ways — both free:

Generate ideas (rules) — instant, deterministic rules on recent prices. No AI, no key.

Claude Code prompt — AI-written ideas using your Claude subscription: copy the prompt, paste it into Claude Code, and it writes the ideas to your Drive. No API key needed.
